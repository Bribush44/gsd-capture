# GSD Capture Friend Beta Launch Checklist

## Before sharing

- [ ] Open the shared Apple Shortcut on Brian's iPhone.
- [ ] Rename the shared template to `Capture`.
- [ ] Confirm the shared template contains no personal Voice Key.
- [ ] Confirm the import question asks each tester for their own Voice Key.
- [ ] Share the renamed Shortcut and copy the new iCloud link.
- [ ] Replace the current iCloud Shortcut URL in `index.html` if the link changed.
- [ ] Test onboarding in a clean/private Safari session.
- [ ] Confirm the new account only sees its own GSD Review items.
- [ ] Confirm the new account only writes to its own Microsoft To Do and Calendar.
- [ ] Confirm the Beta Tester Guide opens from the app.
- [ ] Create a Git tag after Production is approved.

## Minimum friend test

1. Install GSD Capture on the Home Screen.
2. Connect Microsoft To Do.
3. Set up Voice Capture.
4. Install the Shortcut with the tester's private key.
5. Test a normal task.
6. Test an AI folder suggestion and approval.
7. Test an explicit reminder.
8. Test an explicit calendar event.
9. Send feedback using the template in the guide.
