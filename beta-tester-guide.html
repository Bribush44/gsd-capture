GSD Capture Tester Guide

Set it up once, then capture tasks, reminders, and calendar events by voice.

What you need

  • An iPhone with Apple Shortcuts.
  • A personal or work Microsoft account with Microsoft To Do and Calendar access.
  • An internet connection during setup and voice capture.
Keep your Voice Capture key private. Do not send it in a screenshot, text message, or feedback report.

First-time setup

1
Open GSD Capture in Safari
Go to https://gsd-capture.vercel.app
2
Add it to the Home Screen
Tap Share, choose Add to Home Screen, then tap Add.
3
Connect Microsoft To Do
Open Settings in GSD Capture, tap Connect Microsoft To Do, sign in, and accept the requested permissions.
4
Set up Voice Capture
Tap Set Up Voice Capture and complete the Microsoft approval.
5
Copy your private key
Tap Copy Voice Key. The key belongs only to your setup.
6
Install the shared Shortcut
Tap Install Capture Shortcut and paste the key when the Shortcut asks for it. During beta, the shared Shortcut may initially display as “GSD Voice Capture”; it can be renamed to “Capture.”
7
Run one test
Say: “Get a quote from the packaging supplier.” Then check GSD Review and Microsoft To Do.
Open GSD Capture

Four beta tests

Say thisExpected result
“Capture buy printer paper.”A task appears in GSD Review and Microsoft To Do.
“Get a quote from the packaging supplier.”GSD suggests Purchasing, shows confidence and an explanation, and waits for approval.
“Remind me tomorrow at 8 AM to call the supplier.”The task has a Microsoft To Do reminder for the requested time.
“Add a dentist appointment Friday at 2 PM for one hour to my calendar.”A Microsoft calendar event is created and appears in Apple Calendar when that Microsoft calendar is enabled on the iPhone.

Reviewing a capture

  1. Open GSD Capture and choose Review.
  2. Tap Refresh if the new capture is not visible yet.
  3. Read the suggested folder, confidence, and explanation.
  4. Approve the suggested folder, choose another folder, or keep it in GSD Review.
  5. Confirm the task moved to the correct Microsoft To Do list.

Troubleshooting

The Shortcut says the key is invalid

Open GSD Capture Settings, copy the Voice Key again, and replace the x-gsd-capture-key value inside the Shortcut.

The task is in Microsoft To Do but not GSD Review

Confirm Voice Capture shows Connected in the same GSD app. Then open Review and tap Refresh.

Microsoft is disconnected

Reconnect Microsoft To Do, then choose Set Up Again under Voice Capture and update the Shortcut with the new key.

The calendar event is not in Apple Calendar

Open iPhone Settings, go to Apps > Calendar > Calendar Accounts, select the same Microsoft account, and enable Calendars.

Still stuck

Open Setup help and recovery in GSD Settings and tap Copy Safe Support Details. Send those details with a screenshot that does not show the Voice Key.

Current beta limitations

  • Voice capture requires an internet connection.
  • Calendar events are created in Microsoft Calendar and sync to Apple Calendar when that account is enabled.
  • Recurring schedules and complex repeating reminders are not yet the focus of this beta.
  • The typed task box is available, but this beta is primarily testing the iPhone voice Shortcut.
